# Takakuda
## Installation

Add this line to your application's Gemfile:

```ruby
gem 'takakuda'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install takakuda

## Usage
All object can be 'Takakuda'

```ruby
require 'takakuda'
'hoge'.to_takk
1.to_takk
[1, 2, 3].to_takk
```
All of them are changed to 'Takakuda'
